# WeatherApp
Key concepts to build this app:
- Asynchronous code
- Promises
- Promise chaining
- Async and await
- Work with APIs (in this case OpenWeatherMap API)

 Future upgrades:
 - Display all the weather information for the next four days starting from the users request day
 - Those days could hid the information related to the weather and display it with some short of animation like flashcards or accordion animation
